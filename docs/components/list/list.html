List Group

Use the .list-group class to create a stacked list. To add item separators, add the .list-separators class.

  • Roronoa Zoro

  • Itachi Uchiha

  • Gojo Satoru

  • Monkey D Luffy

  • Namikaze Minato

  • Madara Uchiha

            
            <ul class="list-group">
                <li class="list-group-item">
                    <div class="avatar bg-default avatar-sm">
                        <i class="fas fa-user"></i>
                    </div>
                    <p class="text-base">Roronoa Zoro</p>
                </li>
                ...
            </ul>

            <!-- w/ List item separators -->
            <ul class="list-group list-separators">
                <li class="list-group-item"> ... </li>
                ...
            </ul>
